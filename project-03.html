Web Experience Design

Type UX / UI
Platform Web
Overview

A complete web experience for a creative studio seeking to relaunch their digital presence. The project encompassed UX strategy, interaction design, visual design, and front-end prototyping — from blank canvas to shipped product.

The core challenge: how do you communicate craft and care through a screen? The answer was in the details — hover states that delight, transitions that feel inevitable rather than mechanical, and a layout that rewards exploration.

User research, competitive landscape analysis, and iterative prototyping informed every decision. The result was a 40% improvement in average session duration and a significant reduction in bounce rate.
